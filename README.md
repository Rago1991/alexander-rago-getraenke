# Alexander Rago Getränkeabrechnung

One-Pager und Brandguide für eine Getränkeabrechnungssoftware für Unternehmen.

## Dateien

- `index.html` – Marketing-One-Pager
- `brandguide.html` – Brandguide mit Farben, Typografie, Tonalität und UI-Stil
- `.nojekyll` – GitHub-Pages-Kompatibilität

## GitHub Pages

Aktivierung über:

Settings → Pages → Deploy from a branch → main → / root → Save
